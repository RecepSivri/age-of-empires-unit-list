## Run Code
1. clone this code your locale.
2. install angular version 12 here https://v12.angular.io/guide/setup-local
3. then enter this project folder and run npm install
4. run ng serve command and open localhost:4200


## Run Test
1. run ng test --code-coverage command
2. Open index.html file from coverage folder.
